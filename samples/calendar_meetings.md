## 등록 방법 (운영진용)

### 옵션 A — 수동 등록 (참가자 수 적을 때)
1. 각 참가자 계정으로 로그인
2. Calendar → 새 일정 → 위 표대로 입력
3. 참석자 이메일은 워크샵용 더미 이메일 (예: `nvidia-mgr@example.com`) 사용

### 옵션 B — Apps Script 일괄 등록 (권장, 30명+ 참가자)
운영진이 Google Apps Script로 위 6개 일정을 모든 참가자 캘린더에 자동 등록. 별도 스크립트 문의.

### 옵션 C — Calendar API 스크립트
Google Workspace Admin 권한으로 도메인 전체에 일정 일괄 push. 참가자 계정이 같은 Workspace 도메인일 때만 가능.

## 워크샵 진행 흐름과의 연결

이 사전 등록 미팅은 핸즈온 **에이전트 #1 "내일 미팅 사전 브리핑"** 시나리오의 입력 데이터입니다.

참가자가 에이전트에 *"내일 미팅 사전 브리프 만들어줘"* 라고 요청하면:
1. **Calendar 호출** → 위 6개 미팅 자동 인식
2. **각 회사 식별** — NVIDIA Korea / 삼성전자 / 네이버 / Microsoft Korea / SK하이닉스 / 셀트리온
3. **DART MCP 호출** — 4개 상장사는 풀 분석, 2개 외국계는 폴백 메시지
4. **사내 매뉴얼 적용** (데이터스토어) — 미팅 사전 체크리스트
5. **Drive 저장** — `미팅 브리프 [날짜]` 폴더에 회사별 1쪽 문서
6. **Gmail 초안** — 본부장 보고용 (선택)

청중은 *"엔비디아·삼성·네이버 미팅 자동 분석"* 의 임팩트와, *"외국계는 DART 안 되는구나"* 같은 한계 인식을 동시에 경험합니다.
